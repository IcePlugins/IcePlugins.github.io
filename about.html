Unturned Plugins at IcePlugins

Free and open source extensions for your Unturned/RocketMod server.


IcePlugins is an Unturned plugin repository created by ExtraConcentratedJuice. My mission is to deliver free, open source plugins for Unturned servers so that you don't have to pay $5 for 20 lines of code.

All of the plugins listed on this website are my creations.

If you would like to request an Unturned plugin or report a bug, you can join the Discord.
